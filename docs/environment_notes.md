# Development Environment Notes

This project was developed as part of a practical assignment in the course **Advanced Cryptographic Implementations** at the Technical University of Munich.

The assignment focused on implementing cryptographic algorithms in a realistic embedded environment rather than as standalone programs.

---

## Provided Development Environment

Students were provided with a **preconfigured VirtualBox development image** containing a complete embedded development toolchain.

The environment included:

* OpenTitan firmware framework
* Ibex RISC-V processor core
* QEMU simulation environment
* UART logging utilities
* build system and firmware tooling

This setup allowed students to implement cryptographic primitives in a realistic hardware-oriented software stack.

---

## Project Integration

The AES masking implementation was integrated into the OpenTitan firmware structure.

The program entry point (`main`) runs inside the OpenTitan runtime environment and uses:

* device configuration libraries
* UART logging infrastructure
* runtime support for the Ibex processor

The code computes the masked AES S-box and prints validation output via UART.

---

## Scope of This Repository

The full OpenTitan environment contains thousands of files and generated artifacts. Only the files that were directly implemented or modified for the coursework are included here.

These include:

* the masked AES S-box implementation
* the masking arithmetic functions
* constant tables used for finite-field arithmetic

All other files from the original environment are intentionally omitted.

---

## Why the Environment Is Not Included

The original development environment includes:

* large external codebases
* autogenerated firmware components
* simulator infrastructure
* build artifacts

Including the entire environment would make the repository unnecessarily large and obscure the actual implementation work.

Instead, this repository focuses on the **core cryptographic implementation written during the coursework**.
